## 5. Mobile Responsive Notes

All wireframes adapt to mobile with:
- Hamburger menu with slide-out navigation
- Single column layouts
- Stacked cards (horizontal → vertical)
- Sticky "Make a Referral" button at bottom
- Touch-friendly tap targets (min 44px)
- Reduced hero text, larger CTAs
- Collapsible sections to reduce scroll depth
