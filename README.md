# MCR Educational

**MCR HQ CIC - Educational Platform**

> Empowering Young People Through Dance, Education & Support

The dedicated educational arm of MCR HQ CIC, providing alternative education provision for young people in Greater Manchester.

## Tech Stack

- **Frontend:** Next.js 15 (App Router) + TypeScript
- **Styling:** Tailwind CSS v4 + shadcn/ui
- **CMS:** Payload CMS v3
- **Database:** PostgreSQL
- **Hosting:** Vercel + Railway

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build
```

## Project Structure

```

│   ├── payload/        # Payload CMS collections & config
```
